# Техническое задание: WebApp для майнинг-пула с реферальной программой

## 1. Общее описание

Веб-приложение для коллективного майнинг-пула токенов GNK с реферальной программой, управлением кошельками и инвестициями в пулы. Платформа **GonkaOne** предоставляет пользователям возможность коллективно инвестировать в майнинг токенов GNK через децентрализованную сеть Gonka.

---

## 2. Дизайн и визуальная концепция

### 2.1. Цветовая палитра (Ультра современная криптовая с неоном)

**Основные цвета:**
- **Фон**: Глубокий черный (#000000) / Темно-серый (#0A0A0A)
- **Акцентный неон**: Яркий циан (#00FFFF) / Электрический синий (#0080FF)
- **Вторичный неон**: Неоновый фиолетовый (#BF00FF) / Розовый (#FF00FF)
- **Успех**: Неоновый зеленый (#00FF41)
- **Предупреждение**: Неоновый оранжевый (#FF8000)
- **Ошибка**: Неоновый красный (#FF0040)

**Градиенты:**
- Линейные градиенты с неоновыми цветами для акцентов
- Свечение (glow effect) для интерактивных элементов
- Тени с неоновым свечением

**Типографика:**
- Минималистичные шрифты (Inter, Space Grotesk, или аналогичные)
- Белый текст (#FFFFFF) на темном фоне
- Неоновые акценты для важной информации

**Стиль:**
- Минимализм с акцентом на функциональность
- Неоновые подсветки для активных элементов
- Плавные анимации и переходы
- Glassmorphism эффекты для карточек

## 5. Раздел "Майнинг"

### 5.1. Статистика майнинга
- Кнопка "СТАТИСТИКА МАЙНИНГА" (неоновая подсветка)
- Общая статистика сети, активных пулов, общей мощности
- Анимированные графики с неоновыми линиями

### 5.2. Доступные пулы
- Список активных пулов для инвестирования

#### Карточка пула (с неоновой рамкой и свечением):
- **Номер пула**: "**GonkaOne** Pool #25" (неоновый акцент)
- **Прогресс сбора**:
  - Прогресс-бар с неоновым градиентом (циан → фиолетовый)
  - Текущая сумма / Целевая сумма (например, "4,900 USDT / 11,900 USDT")
  - Процент заполнения крупным шрифтом с неоновым свечением (например, "41.18%")
- **Оборудование**:
  - Иконка чипа с неоновой подсветкой
  - Тип оборудования (например, "8xH100")
- **Условия**:
  - "Аренда сервера на 30 дней" (иконка шестеренки)
  - "Дата начала майнинга: ДД.ММ.ГГГГ" (иконка календаря)
- **Инвестирование**:
  - Быстрые кнопки: 50, 100, 250, 500, 1000 USDT (неоновые кнопки с hover-эффектом)
  - Кастомный ввод:
    - Поле ввода суммы с неоновой рамкой
    - Кнопки "-" и "+" с неоновой подсветкой
    - Валидация минимальной суммы
  - Кнопка "Купить" (крупная, с градиентом и свечением)
- **Статус пула** (цветовая индикация):
  - Активный сбор (неоновый зеленый)
  - Сбор завершен (неоновый синий)
  - Майнинг начался (неоновый фиолетовый)
  - Майнинг завершен (неоновый циан)

### 5.3. Мои пулы
- **Заголовок**: "Мои пулы" (неоновый акцент)
- **Всего инвестировано**: Крупная карточка с неоновым свечением, сумма в USDT
- **Участие в пулах**:
  - Список пулов пользователя (карточки с неоновой рамкой)
  - Для каждого пула:
    - Номер пула
    - Сумма инвестиции (неоновый акцент)
    - Дата инвестиции
    - Статус с цветовой индикацией
    - Прогресс майнинга (анимированный прогресс-бар)
    - Ожидаемая дата завершения
    - Ожидаемое вознаграждение в GNK (неоновый акцент)
- Если пулов нет: "Нет пулов" (приглушенный текст)

### 5.4. Фильтры и сортировка
- Фильтры по статусу (активные, завершенные) - неоновые переключатели
- Сортировка по дате, сумме инвестиции
- Анимация при фильтрации
